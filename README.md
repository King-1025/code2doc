### 代码整理工具(code2doc)
---
运行环境：python3

配置：
   code.sh init   ---   for Linux
   
   code.cmd init  ---   for Windows 

基本用法：
   code [-p android|java|...] [-m number] [-o output] <project-path>
   code [-c config-file] <project-path>
   
   eg:
     code -p android -m 20 ./
     code -c temp.json ./
     ...
